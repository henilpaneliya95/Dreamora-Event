# 🎉 Dreamora Event – Event Management System

Dreamora Event is a web-based Event Management System developed to manage event bookings, photography, editography services, payments, and administrative operations in a single platform.

This project was developed as part of an academic project for the **Integrated M.Sc. (Data Science & Analysis)** program at **JG University**.

---

## 🚀 Project Objective

The main objective of this project is to provide an online platform where:
- Customers can book events, photographers, and editography services
- Administrators can manage events, users, payments, and reports
- Company partners can track profit sharing and financial records

---

## 🛠️ Technology Stack

### Frontend
- HTML
- CSS

### Backend

### Database
- MySQL

### Server Environment
- WAMP Server (64-bit)

---

## 👥 User Roles

1. **Administrator**
   - Full control over the system
   - Manage events, photographers, editography, payments, reports, and settings

2. **Customer**
   - Register & login
   - Book events, photography, and editography services
   - Make advance and full payments
   - View invoices and booking status

3. **Company Partner**
   - View profit sharing details
   - Access financial reports

## 💳 Payment System

- Advance payment: **40% mandatory**
- Balance payment after service completion
- Invoice generation supported
- (Note: Real-time payment gateway is not integrated)
